Todo List API Tester

Interactive Swagger UI for local testing of the Todo List Backend API. Use the Authorize button to paste a JWT as a Bearer token, then send real requests directly to http://localhost:8080.

OpenAPI 3.0.1 Swagger UI via CDN JWT Bearer Auth Server: localhost:8080 Spring Boot 3.5.6

Interactive Console

Controllers are grouped as Auth, Tasks, and Health. Public auth endpoints stay open, while protected task operations require Bearer token authorization.
